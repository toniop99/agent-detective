Use the sidebar to browse. **Live (after CI):** [https://toniop99.github.io/agent-detective/](https://toniop99.github.io/agent-detective/) (GitHub project Pages; enable in repo **Settings → Pages** with source **GitHub Actions** if not already).

The **source of truth** for edits is still [`docs/`](https://github.com/toniop99/agent-detective/tree/main/docs) in git; the sync script rewrites in-repo links for the site.
